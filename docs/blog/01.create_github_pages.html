# #1. 创建 Github Pages

参考Github Pages在新窗口打开

# 基础步骤

  • 步骤 1：在 Github 中创建新的 repository，命名为 username.github.io
  • 步骤 2：在新建的 repository 中，添加 index.html 文件
  • 步骤 3：打开https://username.github.io能够打开刚刚编写的index.html文件

# 进阶配置

# 指定默认 branch 和目录？

在 Github 网站上，进入对应的 Github Pages Repository，进入 Settings -> Pages。 可以选择指定的 branch 和目录来配置 GitHub Pages 站点的默认目录。
